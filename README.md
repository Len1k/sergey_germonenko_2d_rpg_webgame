# Deep Dream Protocol — 2D RPG Webgame

Браузерная демо-версия 2D RPG в стиле пиксельного кибер-безумия. Игра запускается как статический сайт и не требует сборки.

## Что уже реализовано

- Вид сверху на Canvas: тайловые стены, ресурсы, враги, боссы, порталы и герой.
- Создание персонажа с именем и одним из трёх архетипов: Джаброни, Нейро-шаман или Крипто-омега.
- Пять слоёв реальности: Офис, Кибер-город, Гачи-Верс, Крипто-Стадион и Задубенный Ангар.
- Пошаговые столкновения с действиями: код-атака, дебафф `try/except`, использование артефакта.
- HUD с HP, волей к коду, опытом, мем-монетами, шкалой S.O.S.I. и прогрессом артефактов.
- Инвентарь, сбор ресурсов, крафт Священного доширака и ритуал сохранения у доширак-точек.
- Случайные мем-инфекции, бегущая новостная лента, отряд союзников и журнал приключения.
- Финальная консоль с четырьмя концовками: Python, C++, Задубение и `merge --squash`.
- Экранные кнопки движения для мобильных устройств.

## Управление

- `WASD` или стрелки — движение.
- `1` — код-атака в бою.
- `2` — дебафф `try/except` в бою.
- `3` — использовать Священный доширак в бою.
- Кнопка «Скомпилировать героя» применяет выбранное имя и архетип.
- Кнопка «Новая симуляция» перезапускает прохождение.

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python3 -m http.server 8000
```

Затем перейдите на `http://localhost:8000`.
