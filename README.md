# Deep Dream Protocol — 2D RPG Webgame

Расширенный статический прототип браузерной 2D RPG по промту «Deep Dream Protocol». Проект остаётся обычным HTML/CSS/JS-сайтом без сборки, но теперь содержит каркас большой кампании: акты, ветки, союзников, крафт, достижения, сохранения и New Game+. Контент вынесен в отдельный `data.js`, а движок остался в `script.js`.

## Масштаб кампании

Целевой дизайн — большая RPG на 35–50 часов основного сюжета и дополнительный сайд-контент. В текущей статической версии реализован **играбельный вертикальный срез** этих систем: все ключевые механики есть в компактном виде, чтобы их можно было быстро тестировать в браузере и расширять контентом.

## Архитектура

- `data.js` — база контента: архетипы, акты, слои, враги, ресурсы, сайд-квесты, кодекс, атлас, навыки, рейды и достижения.
- `script.js` — игровой движок, обработчики UI, валидация контента, движение, бой, крафт, диалоги, сохранения и отрисовка.
- `CONTENT_GUIDE.md` — правила расширения контента и описание автопроверок.

## Что реализовано

- Вид сверху на Canvas: тайловые стены, ресурсы, враги, боссы, порталы и герой.
- Создание персонажа с именем и одним из трёх архетипов: Джаброни, Нейро-шаман или Крипто-омега.
- Актовая структура кампании: пролог, C++-ивация, выбор стороны, расследование S.O.S.I., спасение FEDIL и финальный центр управления реальностью.
- Пять слоёв реальности: Офис, Кибер-город, Гачи-Верс, Крипто-Стадион и Задубенный Ангар.
- Ветвление в Акте 2: Python/FEDIL, C++/Гермоненко или танковая ветка Глеба.
- Пошаговые столкновения с действиями: код-атака, дебафф `try/except`, использование артефакта.
- HUD с HP, волей к коду, опытом, мем-монетами, шкалой S.O.S.I., актом, веткой и прогрессом артефактов.
- Инвентарь, сбор ресурсов, крафт Священного доширака и ритуал сохранения через localStorage.
- Диалоги с уровнем доверия/безумия, лагерь Ярика с чат-ответами и синтаксические дуэли через ввод команд.
- Доска сайд-квестов для каждого слоя: сдача ресурсов, награды опытом/монетами, доверие и дополнительные артефакты.
- Атлас слоёв с районами, рейдовыми зонами и сюжетными узлами для планирования большой карты.
- Дерево навыков с очками за боссов/сайд-квесты и веточными пассивками Python/C++/танков.
- Рейдовые операции: рискованные вылазки за редким лутом, репутацией фракций и артефактами.
- Кодекс мира с краткими справками по S.O.S.I., персонажам, дошираку и New Game+.
- Переключаемый текстовый саундтрек по слоям для прототипирования музыкального направления.
- Случайные мем-инфекции, кликабельная пинг-тревога, бегущая новостная лента, отряд союзников и журнал приключения.
- Достижения: первый бой, спасение FEDIL, S.O.S.I.-выживание, танковые ключи, доширак, синтаксис и открытие New Game+.
- Финальная консоль с четырьмя концовками: Python, C++, Задубение и `merge --squash`.
- New Game+ «Бесконечная шиза» с отдельным слоем и усиленным стартом.
- Экранные кнопки движения для мобильных устройств.
- Автовалидация контента при старте: карты, порталы, боссы, ID квестов/рейдов/навыков и базовые награды проверяются до запуска игры.
- Dev-панель для контент-дизайнера: телепорт по слоям, выдача ресурсов/опыта/очков навыков, закрытие целей слоя и экспорт/импорт сейва.

## Управление

- `WASD` или стрелки — движение.
- `1` — код-атака в бою.
- `2` — дебафф `try/except` в бою.
- `3` — использовать Священный доширак в бою.
- `T` — открыть диалог/цитатник.
- `Y` — синтаксическая дуэль.
- `C` — лагерь Ярика.
- `Q` — доска сайд-квестов.
- `O` — атлас слоёв.
- `K` — дерево навыков.
- `L` — рейдовые операции.
- `B` — кодекс мира.
- `M` — включить/выключить текстовый саундтрек.
- `` ` `` — открыть Dev-панель контент-дизайнера.
- Кнопка «Скомпилировать героя» применяет выбранное имя и архетип.
- Кнопка «Новая симуляция» перезапускает прохождение.
- Кнопка «Загрузить доширак-сейв» поднимает сохранение из `localStorage`.

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python3 -m http.server 8000
```

Затем перейдите на `http://localhost:8000`.
